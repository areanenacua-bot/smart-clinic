Installation Guide

Since this is a thesis project app, follow these steps to bypass Android security blocks.

1

Download the APK

Click the download button above to get the app-debug.apk file.

2

Allow Installation

If Android blocks the file, go to your Settings and enable "Allow from this source" for your browser/file manager.

3

Bypass Play Protect

On the "Blocked by Play Protect" screen, click More Details and then Install Anyway.

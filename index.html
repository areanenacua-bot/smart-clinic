First-Time Setup

A few permissions make the app work properly during emergencies and clinic updates.

Location Permission

Enable location when using SOS. Responders use GPS coordinates and campus landmarks to identify where help is needed.

Notification Permission

Allow notifications so you can receive announcements, appointment updates, and emergency response status messages.

Account Details

Use the account provided or approved by the system administrator. Keep profile and contact information accurate.

Internet Connection

The app sends emergency alerts and receives updates through Firebase, so mobile data or Wi-Fi should be available.

Using SOS Responsibly

The SOS button is for urgent situations on campus where immediate clinic or ERT assistance may be needed.

WHEN

Use it for fainting, severe injury, breathing difficulty, seizure, serious allergic reaction, or similar emergencies.

HOW

Hold the SOS button until the app confirms it is sending. Stay in a visible and safe area if possible.

AFTER

Wait for response updates. The app may show that ERT is responding or that the case has been resolved.

Troubleshooting

Most issues are caused by permissions, connection, or outdated app files.

SOS says location is unavailable

Turn on phone GPS, allow location permission for Smart Clinic, then wait a few seconds for accuracy to improve.

iOS: How do I save the web app to my home screen?

Open the iOS Web App Link using Apple Safari. Tap the Share button (the square icon with an arrow pointing up), scroll down, and select "Add to Home Screen". It will now launch with full-screen focus like a normal app.

No notifications are showing

Check device notification permissions, battery saver restrictions, and internet connection. Log out and log in again to refresh your device token.

Login is not accepted

Confirm your email and password with the clinic or system administrator. Inactive accounts cannot access the system.

App looks outdated

Ask for the latest release from the project team. Some features require the updated APK and deployed Firebase functions.
